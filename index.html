3. Verarbeitete Daten

  • Standortdaten beim Erstellen einer Meldung
  • Fotos und Texte, die Sie im Rahmen einer Meldung übermitteln
  • Geräte-Informationen (z.B. Betriebssystem, App-Version)
  • Benutzername oder Name (sofern bei der Nutzung angegeben)

4. Zweck der Verarbeitung

Die Verarbeitung erfolgt zum Zweck der Bearbeitung und Verwaltung Ihrer Meldungen sowie zur Verbesserung der Weginfrastruktur in der Region Achensee.
Die Erfassung des Namens erfolgt zur Zuordnung und Bearbeitung eingereichter Meldungen sowie zur eventuellen Rückfrage.

5. Speicherort der Daten

Die Daten werden aktuell in einem Appwrite Cloud-System gespeichert. Zu einem späteren Zeitpunkt erfolgt die Migration auf einen eigenen Server innerhalb der EU.

6. Weitergabe der Daten

Es erfolgt keine Weitergabe Ihrer Daten an Dritte. Die Daten werden ausschließlich vom Tourismusverband Achensee bzw. beauftragten Dienstleistern verarbeitet.

7. Ihre Rechte

Sie haben das Recht auf Auskunft, Berichtigung, Löschung, Einschränkung, Datenübertragbarkeit sowie Widerspruch. Bei Fragen wenden Sie sich bitte an info@achensee.com.
